React Web Component Example

Creating a web component with React.js

Below is a web component. If you inspect the html on this page, you'll see its tag as <web-component>
You can include this component in your own HTML by loading this script in your <head> tag
<script src=""></script>
Then place this custom HTML tag in you markup
<head>
For Example 
<!DOCTYPE html>
<html>
  <head>
    <script src=""></script>
  </head>
  <body>
    <web-component></web-component>
  </body>
</html>
View the source code at https://github.com/Sirrine-Jonathan/react-web-component
